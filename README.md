# 🌟 포트폴리오 웹사이트 (Noh-je's Website)

## 🚀 프로젝트 소개

이 프로젝트는 개인 포트폴리오를 소개하기 위해 제작된 반응형 웹사이트입니다. HTML, CSS, JavaScript를 사용하여 구현되었으며, 사용자 경험(UX)을 향상시키기 위한 다양한 상호작용 기능을 포함하고 있습니다.

## ✨ 주요 기능 (Features)

웹 표준을 준수하며 사용자 친화적인 경험을 제공하기 위해 다음과 같은 상호작용 기능을 구현했습니다.

### 1. 동적 내비게이션 및 스크롤 기능

- **부드러운 스크롤 (Smooth Scroll):** 상단 **내비게이션 메뉴** 클릭 시 해당 섹션으로 부드럽게 이동합니다.
- **동적 하이라이트:** **Intersection Observer API**를 활용하여 현재 뷰포트에 표시되는 섹션에 따라 상단 내비게이션 메뉴의 하이라이트가 동적으로 변경됩니다.
- **TOP 버튼:** 페이지 우측 하단의 **TOP 버튼** 클릭 시 페이지 맨 위로 부드럽게 이동합니다.

### 2. 사용자 인터페이스 설정

- **다크 모드 / 라이트 모드 전환:** 헤더 로고 옆에 위치한 **테마 전환 버튼**을 클릭하여 페이지의 전체적인 색상 테마를 다크 모드와 라이트 모드로 전환할 수 있습니다. (CSS 변수와 JavaScript의 `classList.toggle`을 사용하여 구현)

## 🛠 기술 스택 (Tech Stack)

- **HTML5**
- **CSS3** (반응형 웹 디자인 포함)
- **JavaScript (ES6+)**
- **Intersection Observer API** (동적 내비게이션 구현)

## 📌 파일 구조

- `index.html`: 웹사이트의 메인 구조 및 내용
- `css/style.css`: 웹사이트의 스타일 정의 (다크/라이트 모드 변수 포함)
- `main.js`: 상호작용 및 동적 기능을 구현하는 JavaScript 코드 (모듈 방식으로 로드)
